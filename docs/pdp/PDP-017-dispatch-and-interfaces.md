# PDP-017: Dispatch and Interfaces

**Status:** Draft
**Issues:** #30

## Summary

Add open dispatch (multimethods) to Pipe. A `dispatch fn` declares a function signature with untyped dispatch parameters; concrete implementations supply types for those parameters. The runtime selects the matching implementation based on argument types, supporting single- and multi-dispatch.
